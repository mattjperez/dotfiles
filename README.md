My personal dotfiles.

requirements:

- neovim >= 0.8.1
  - ubuntu: `sudo add-apt-repository ppa:neovim-ppa/unstable; sudo apt update; sudo apt install neovim`
- curl
- build-essential
- pkg-cfg
- libssl
- cmake
- npm

Includes:

- git
- zsh
- neovim

```
$ git clone https://github.com/mattjperez/dotfiles.git
$ cd dotfiles
$ ./setup.sh
```
